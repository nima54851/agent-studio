# 🔥 GitHub 自动化 Pro — 产品说明

> 进阶版 GitHub AI 自动化：PR 合并策略 / 代码质量门禁 / 自动发布 / Issue 追踪
> 让你的 GitHub 仓库真正自动化运转

---

## 产品概述

进阶版 GitHub 自动化 Agent，在入门版基础上增加了：

| 功能 | 详情 |
|---|---|
| 🤖 PR 自动合并 | 设置条件规则，符合则自动合并（通过 CI + 2人 review） |
| 🔒 代码质量门禁 | PR 必须通过 AI 审查 + 测试覆盖率 + lint 才可合并 |
| 🚀 发行版管理 | 依据 milestone 自动生成 changelog，发布 Release |
| 🏷️ Issue 自动化 | 自动分类、贴标签、分配负责人、追踪进度 |
| 📊 仓库周报 | 每周自动生成仓库活跃度报告 |
| 🤝 Code Review 分配 | 根据代码改动范围智能分配 Reviewer |
