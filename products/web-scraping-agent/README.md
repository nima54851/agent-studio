# 🕷️ AI 网页数据爬取 Agent — 产品说明

> 说出要爬的网站和数据字段，AI 自动构建爬取流程
> 支持登录/翻页/动态渲染，输出 CSV / JSON / 数据库

---

## 产品概述

这是一个**智能网页爬取 Agent**，基于 OpenClaw + n8n 构建。

| 功能 | 详情 |
|---|---|
| 🗣️ 自然语言爬取 | "帮我爬这个网站的所有产品价格和评分" |
| 🔐 登录/认证 | 支持需要登录的页面（Cookie/Session） |
| 📄 翻页处理 | 自动识别并遍历分页 |
| ⚡ 动态渲染 | 支持 JavaScript 渲染页面（Playwright） |
| 📊 格式输出 | 支持 CSV / JSON / Google Sheets / 数据库 |
| ⏰ 定时任务 | 支持定时爬取，数据自动更新 |
| 🛡️ 合规检测 | 自动检测 robots.txt，规避法律风险 |
