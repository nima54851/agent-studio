# ⚙️ n8n 工作流构建 Agent — 产品说明

> 说出你想要的工作流，AI 自动生成 n8n JSON 配置
> 无需记忆节点，描述需求 → 生成工作流 → 一键导入

---

## 产品概述

这是一个**自然语言驱动的 n8n 工作流生成 Agent**，基于 OpenClaw 构建。

| 功能 | 详情 |
|---|---|
| 🗣️ 自然语言生成 | 用中文描述需求，自动生成 n8n workflow.json |
| 🔄 模板市场 | 50+ 预设模板（社交媒体/CRM/自动化办公/数据处理） |
| 🔧 一键导入 | 生成的 JSON 直接导入 n8n |
| 🔍 错误修复 | 工作流运行报错？发给 Agent 自动诊断修复 |
| 📤 导出分享 | 生成可分享的工作流链接 |

---

## 产品包含

- ✅ `agent-skill/` — OpenClaw n8n Builder Skill
