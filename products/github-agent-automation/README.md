## 产品概述

这是一套**完整运行的社交媒体自动运营 Agent 系统**，基于 n8n 构建，每天自动执行以下操作：

| 功能 | 详情 |
|---|---|
| 🔍 GitHub 热点追踪 | 自动抓取当天 AI/ML 热门项目 |
| ✍️ 智能评论生成 | AI 生成高质量技术评论（不是模板套话） |
| ⭐ 自动 Star | 每天 Star 1-2 个优质项目 |
| 📊 数据报告 | 生成每日 AI 资讯报告 |
| 📡 Webhook 分发 | 推送报告到任意平台（Telegram / 微信 / Email） |
