# Git Automation Skill

> AI-powered Git workflows: smart commit messages, automated changelog generation, branch management, PR creation, and repository housekeeping.

## What This Does

Automates common Git operations using AI: generate meaningful commit messages from diffs, auto-generate CHANGELOGs, manage branches at scale, create PRs with descriptions, detect and clean up stale branches, and run repository health checks.

## Core Capabilities

### AI-Powered Commit Messages
- **Semantic commit types**: feat, fix, docs, style, refactor, test, chore
- **Conventional Commits**: Auto-formatted with scope and breaking change detection
- **Multi-commit batching**: Group related changes into logical commits
- **Emoji support**: Optional emoji prefix for better visual parsing

### Automation
- **Changelog generation**: Auto-generate CHANGELOG.md from commit history (Keep a Changelog format)
- **Branch management**: Auto-create feature branches, detect stale branches, auto-merge safe changes
- **PR automation**: Auto-create PRs with AI-generated descriptions, reviewers, labels
- **Repository health**: Detect large files, unused branches, uncommitted secrets

### Multi-Repo Support
- **Repo scanning**: Scan org for inactive repos, outdated dependencies
- **Cross-repo operations**: Tag, branch, or cherry-pick across multiple repos
- **Template repos**: Auto-apply `.github/templates/` to new repos

## Usage

```bash
# Generate commit message
python3 commit_message.py --diff-file /tmp/diff.patch

# Generate commit message for staged changes
git diff --cached | python3 commit_message.py --stdin

# Generate changelog
python3 changelog_gen.py --repo . --output CHANGELOG.md

# Check repo health
python3 repo_health.py --repo .

# n8n workflow
# Import n8n-git-automation.json into your n8n instance
```
