# Cover Image Prompts

## 博客封面图

```
{style} style cover image for article: {TOPIC}
High quality, 16:9 aspect ratio, vibrant professional,
digital art, cinematic lighting, ultra detailed, 4k
```

Styles: `Cinematic`, `Minimalist`, `Futuristic`, `Watercolor`, `3D Render`, `Cyberpunk`, `Vintage`, `Editorial`

## 社交媒体配图

```
Professional social media post image about: {TOPIC}
{STYLE}, clean design, {ASPECT_RATIO} aspect ratio,
modern, eye-catching, suitable for {PLATFORM}
```

Platforms: Twitter (1200x675), Instagram (1080x1080), LinkedIn (1200x627), YouTube (1280x720)
