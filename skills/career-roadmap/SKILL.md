# Career Roadmap

## 技术方向总览

```
前端开发 ──→ React/Vue + 工程化 ──→ 全栈
后端开发 ──→ Go/Java/Node ──→ 分布式 ──→ 架构
AI/ML ──→ Python ──→ 模型训练 ──→ LLM应用
DevOps ──→ K8s ──→ 平台工程 ──→ SRE
移动端 ──→ Flutter/React Native ──→ 跨平台
```

## 阶段式成长路线

### 🟢 阶段1：初学者（0-1年）
**目标**：能独立完成功能模块

| 方向 | 必学 | 建议 |
|---|---|---|
| 前端 | HTML/CSS/JS → React 基础 | 做个 Todo App |
| 后端 | Node.js/Python → REST API | 做个博客后端 |
| 数据库 | SQL 基础 → MySQL | 学会 CRUD |
| 工具 | Git → Docker 基础 | 能部署简单项目 |

**推荐项目**：个人博客、天气 App、命令行工具

---

### 🟡 阶段2：中级工程师（1-3年）
**目标**：能独立负责完整功能

| 方向 | 必学 | 建议 |
|---|---|---|
| 前端 | TypeScript → 状态管理 → 性能优化 | 重构之前的项目 |
| 后端 | 数据库设计 → 缓存(Redis) → 消息队列 | 学会选型 |
| DevOps | Docker Compose → CI/CD → K8s 入门 | 自动化部署 |
| 工程化 | 设计模式 → 重构 → 测试(TDD) | 提升代码质量 |

**推荐项目**：电商后端、物联网数据平台、AI 对话应用

---

### 🔴 阶段3：高级工程师（3-5年）
**目标**：能带技术方案，能指导他人

| 方向 | 必学 | 建议 |
|---|---|---|
| 架构 | 分布式系统 → 微服务 → 事件驱动 | 设计高并发系统 |
| 云 | AWS/GCP → Terraform → 成本优化 | 学会选云方案 |
| 安全 | OAuth2 → JWT → 渗透测试基础 | 重视安全设计 |
| 软技能 | 技术方案文档 → code review → 跨团队沟通 | 培养技术影响力 |

**推荐项目**：微服务改造、高并发系统设计、技术分享博客

---

### 🟣 阶段4：技术专家/架构师（5年+）
**目标**：制定技术战略，影响业务方向

- 系统架构设计与评审
- 技术债务管理
- 团队技术人才培养
- 技术标准制定

## 面试准备方案

### 算法准备（LeetCode 路线）
```
第1周：数组、链表、哈希表（15题）
第2周：栈、队列、递归（10题）
第3周：二分查找、滑动窗口（10题）
第4周：二叉树、深度优先（15题）
第5周：动态规划（入门10题）
第6周：图论、高级 DP（10题）
```

### 系统设计准备
1. 设计短链接服务（TinyURL）
2. 设计 Twitter/微博时间线
3. 设计聊天系统（WebSocket）
4. 设计搜索建议系统
