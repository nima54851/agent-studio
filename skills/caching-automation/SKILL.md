# Caching Automation Skill

> AI-powered cache strategy design, implementation, and optimization for distributed systems.

## What This Does

Design and implement intelligent caching layers across Redis, Memcached, and CDN. Covers cache-aside, write-through, write-behind patterns, TTL optimization, cache warming, invalidation strategies, and distributed lock patterns.

## Core Capabilities

### Cache Strategy Design
- **Pattern selection**: cache-aside, write-through, write-behind, read-through
- **TTL optimization**: ML-based TTL estimation from access patterns
- **Key design**: Naming conventions, namespace isolation, versioned keys
- **Invalidation**: Time-based, event-based, version-based

### Implementation
- **Redis**: String, Hash, Sorted Set, Stream caching
- **Memcached**: Simple value caching with compression
- **CDN**: Edge caching with stale-while-revalidate
- **Local LRU**: In-process cache with size limits

### Monitoring & Optimization
- **Hit rate analysis**: Track hit/miss/eviction rates
- **Cold start prevention**: Cache warming strategies
- **Cache stampede prevention**: Probabilistic early expiration, distributed locks
- **Cost tracking**: Memory usage per cache key family
