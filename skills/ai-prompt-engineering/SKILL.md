# AI Prompt Engineering Skill

> Systematic prompt design, testing, versioning, and optimization for production AI systems.

## What This Does

Provides a structured framework for designing, testing, versioning, and optimizing prompts across the full lifecycle. Covers system prompts, few-shot examples, chain-of-thought reasoning, multi-turn conversations, and prompt A/B testing.

## Core Capabilities

### Prompt Design
- **Anatomy analysis**: System prompt → context → few-shot → user input → output format
- **Role prompting**: Clear persona definitions with constraints
- **Few-shot learning**: Optimal example selection and formatting
- **Chain-of-thought**: Structured reasoning patterns
- **Output formatting**: JSON schemas, markdown, structured data

### Prompt Testing & Versioning
- **A/B testing**: Statistical comparison of prompt variants
- **Regression testing**: Detect quality degradation on prompt changes
- **Evals suite**: Automated quality scoring (accuracy, hallucination, coherence)
- **Version control**: Full prompt git history with diffs

### Optimization
- **Token budget optimization**: Reduce cost while maintaining quality
- **Temperature/top-p tuning**: Finding optimal sampling parameters
- **Context window management**: Truncation strategies, relevance scoring
- **Prompt compression**: Remove redundancy without losing information

## Files in This Skill

## Usage

```bash
# Build a prompt
python3 prompt_builder.py --task "code-review" --lang "python" --output prompts/code-review.md

# Evaluate prompts
python3 prompt_evaluator.py --prompts prompts/v1.md prompts/v2.md --test-set test_cases.json

# Version control
python3 prompt_versioner.py log
python3 prompt_versioner.py diff v1 v2
python3 prompt_versioner.py rollback code-review v3

# n8n workflow
# Import n8n-prompt-testing-workflow.json into your n8n instance
```
