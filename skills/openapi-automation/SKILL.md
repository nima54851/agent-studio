# OpenAPI Automation Skill

> AI-powered OpenAPI specification generation, validation, and documentation.

## What This Does

Automatically generate OpenAPI 3.0/3.1 specifications from code, validate existing specs, and produce interactive documentation (Swagger UI, Redoc). Works with Express, FastAPI, NestJS, and any framework that emits API metadata.

## Core Capabilities

### OpenAPI Generation
- **Code-first**: Scan Express routes, FastAPI decorators, NestJS controllers
- **Schema inference**: Auto-detect DTOs, query params, auth requirements
- **Standard compliance**: Validates against OpenAPI 3.1 spec
- **Multi-format output**: YAML or JSON

### Documentation Generation
- Interactive Swagger UI (`swagger-ui-dist`)
- Redoc static documentation
- Postman collection export
- Mock server generation

### Validation & Quality
- Breaking change detection between versions
- Schema validation (required fields, types, formats)
- Security audit (exposed sensitive fields in schemas)
- Examples completeness check

## Usage

```bash
# Generate from Express app
python3 generate_openapi.py --app ./src/app.js --output openapi.yaml

# Generate from FastAPI
python3 generate_openapi.py --app ./src/api.py --output openapi.yaml --framework fastapi

# Validate spec
python3 validate_spec.py openapi.yaml

# Generate docs
python3 generate_docs.py openapi.yaml --out ./docs

# n8n workflow
# Import n8n-openapi-workflow.json into your n8n instance
```
